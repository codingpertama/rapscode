# Portfolio

This is my personal portfolio website, built with modern web technologies to showcase my projects, skills, and experience.

## 🚀 Tech Stack

This project is built using:

- **React.js** (v19) - UI Library
- **Vite** - Frontend Tooling & Bundler
- **Tailwind CSS** (v4) - Utility-first CSS Framework for styling
- **Framer Motion** - Animations and transitions
- **GSAP** - High-performance scroll animations and effects
- **Three.js** - 3D graphics and rendering
- **EmailJS** - Client-side email sending for the contact form

## 📂 Features

- **Interactive UI**: Fluid animations using Framer Motion and GSAP.
- **Responsive Design**: Fully responsive across desktop, tablet, and mobile devices using Tailwind CSS.
- **Contact Form**: Functional contact form powered by EmailJS without needing a backend.
- **Modern Scroll Effects**: Pinned sections and scroll-triggered reveals for a premium editorial feel.

## 🛠️ Installation & Setup

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/portofolio.git
   cd portofolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:5173/` to view the website.

## 📦 Build for Production

To create a production-ready build:
```bash
npm run build
```
To preview the built project:
```bash
npm run preview
```

## 📝 Contact & EmailJS Setup
To get the contact form working, you need to configure EmailJS. Make sure to update the EmailJS `service_id`, `template_id`, and `public_key` in your `Contact` component to match your EmailJS account settings.
